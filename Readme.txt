How to run the Preowned/Used Car Selling  Management System Project Using PHP and MySQL

1.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

2.Open PHPMyAdmin (http://localhost/phpmyadmin)

3.Create a database with name pocmsdb

4.Import pocmsdb.sql file

5.Run the script http://localhost/pocms
